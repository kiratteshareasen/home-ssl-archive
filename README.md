# home-ssl-archive

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态形式存储，不针对任何特定域名或网站，旨在提供一个简洁、可访问的页面集合。

## 目录说明

```
home-ssl-archive/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面所需的静态资源（图片、样式、脚本等）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件对应一个独立的页面，文件名即页面标识。
- `assets/`：页面引用的资源文件，按类型或页面组织存放。

## 页面归档说明

- 所有页面均为静态 HTML，无需后端支持，可直接在浏览器中打开。
- 页面之间彼此独立，不依赖外部服务或动态内容。
- 归档内容可能不定期更新或调整，不保证长期稳定。
